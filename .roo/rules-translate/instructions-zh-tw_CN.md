# 繁体中文 (zh-TW) 翻译指南

## 关键术语

| 英文术语     | 使用 (zh-TW) | 避免 (大陆用词) |
| ------------- | ----------- | ---------------- |
| file          | 檔案        | 文件             |
| task          | 工作        | 任務             |
| project       | 專案        | 項目             |
| configuration | 設定        | 配置             |
| server        | 伺服器      | 服務器           |
| import/export | 匯入/匯出   | 導入/導出        |

## 格式规则

- 在中文和英文/数字之间添加空格："AI 驅動"（不是"AI驅動"）
- 使用繁体中文引号：「範例文字」（不是"範例文字"）
- 使用台湾计算机惯用术语而非大陆术语
